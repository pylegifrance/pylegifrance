## With uv (recommended)

```bash
uv add pylegifrance
```

## With pip

```bash
pip install pylegifrance
```

## Prerequisites

- Python 3.12 or newer.
- A PISTE account —
  [see the procedure](/pylegifrance/en/operations/create-piste-account/).
