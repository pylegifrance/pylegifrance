title: Obtenir un compte PISTE
description: Créer un compte PISTE et souscrire à l'API Legifrance.
sidebar:
  order: 3
---

1. Créer un compte sur [piste.gouv.fr](https://piste.gouv.fr).
2. Souscrire à l'**API Legifrance** depuis le catalogue PISTE.
3. Récupérer le couple `client_id` / `client_secret` dans la console PISTE.

Guide officiel :
[piste.gouv.fr/en/help-center/guide](https://piste.gouv.fr/en/help-center/guide).
