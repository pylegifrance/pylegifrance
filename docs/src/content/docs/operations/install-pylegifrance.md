## Avec uv (recommandé)

```bash
uv add pylegifrance
```

## Avec pip

```bash
pip install pylegifrance
```

## Prérequis

- Python 3.12 ou plus récent.
- Un compte PISTE —
  [voir la procédure](/pylegifrance/operations/create-piste-account/).
